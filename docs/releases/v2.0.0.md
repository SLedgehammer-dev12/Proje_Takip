# Proje Takip Sistemi v2.0.0

## Öne Çıkanlar

- GitHub Release tabanlı güncelleme kontrolü eklendi
- Tek dosya yüklemede proje kodu, adı, türü ve yazı bilgileri düzenlenebilir hale geldi
- Proje türleri merkezileştirildi ve eski kayıtlar normalize edildi
- RAM göstergesi gerçek proses belleğini göstermeye başladı
- SQLite katmanına `busy_timeout`, `quick_check`, `optimize` ve kontrollü kapanış iyileştirmeleri eklendi

## Teknik Notlar

- Update akışı sadece GitHub `Release` üzerinden çalışır
- Release asset'i ile birlikte `SHA256SUMS` veya eşdeğer checksum dosyası yayınlanmalıdır
- Bu sürümde beklenmedik kapanışlara karşı veritabanı dayanıklılığı artırıldı
