# v3.0.1

Yayin tarihi: 2026-04-23

## Ozet

Bu patch surum, revizyon satirinda yazi dokumani mevcut oldugu halde proje veya revizyon dokumani yuklenmemisse alt panelde yazi on izlemesinin calismamasina neden olan UI orkestrasyon hatasini duzeltir. Ayni akis icinde `release/v3.0` cizgisi guvenli fast-forward entegrasyonu ile `main`e alinmistir.

## Degisiklikler

- Revizyon preview sonucu ne olursa olsun, secili revizyona ait yazi preview payload'i once ayarlaniyor.
- Ust revizyon preview'ini temizleyen akis ile alt yazi preview panelini temizleyen akis birbirinden ayrildi.
- Revizyon dokumani yoksa sadece ust panel durum mesaji guncelleniyor; alt panelde yazi dokumani varsa preview render edilmeye devam ediyor.
- `MainWindow` preview orkestrasyonu icin hedefli regression testi eklendi.

## Analiz

Sorun service katmaninda degil, `MainWindow` icindeki sira ve state temizleme mantigindaydi:

- Revizyon dokumani yoksa `_trigger_preview_update()` erken cikiyor ve `_clear_preview()` cagiriyordu.
- `_clear_preview()` hem ust revizyon preview'ini hem de alt yazi preview kuyrugunu sifirliyordu.
- Bu nedenle yazi dokumani veritabaninda bulunsa bile alt panelin render akisi hic baslayamiyordu.

Yeni duzende revizyon preview eksik olsa bile yazi preview state'i korunuyor ve render hattina ulasabiliyor.
