# Proje Takip Sistemi v2.0.5

Bu sürüm, **Otomatik Güncelleme İndirme (App-in-App Update)** fonksiyonundaki bir kod kaybını telafi eden kritik bir yamadır.

## 🐛 Hata Düzeltmeleri

*   **Uygulama İçi Güncelleme Hata Düzeltmesi:** Önceki (v2.0.3/v2.0.4) sürümlerde bulunan yeni bir güncelleme uyarısını onaylayıp indirmek istediğinizde karşılaşılan `İndirme Başarısız: Güncelleme dosyası indirilemedi. Detay: Dosya indirilemedi.` hatası giderildi. Arka plandaki GitHub erişim ve indirme stream mantığı başarıyla uygulandı ve test edildi.

_Güncelleme için `ProjeTakip-v2.0.5-windows-x64.exe` dosyasını indirip doğrudan çalıştırabilirsiniz._
