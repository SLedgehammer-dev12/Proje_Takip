# v3.2.2 (20 Mayıs 2026)

## Öne Çıkanlar
- **Red Flag Sistemi Geliştirmesi:** Revizyon işaretleme artık sebep, kullanıcı ve tarih bilgisiyle kaydediliyor.
- **Red Flag Dashboard:** Ana sekme çubuğunda yeni "🚩 Red Flag" sekmesi ile tüm işaretli revizyonların tek tabloda görüntülenmesi.
- **Gelişmiş Filtreleme:** "Kırmızı Bayrak (Red Flag)" filtresi ile projelerin flag durumuna göre filtrelenmesi.

## Teknik Detaylar
- Veritabanı şemasına `flag_reason`, `flag_date`, `flag_user` kolonları eklendi (idempotent migration).
- Yeni `dialogs/red_flag_dialog.py` ile sebep giriş dialogu.
- Yeni `ui/panels/red_flag_panel.py` ile dashboard paneli.
- `filters.py`'ye `kirmizi_bayrak` BOOLEAN filtresi eklendi.
- Canlı arama (3 farklı noktada) red flag anahtar kelimelerini destekliyor.
- 5 adet kritik bug düzeltmesi (report butonu, dead code, sinyal bağlantıları, bellek sızıntısı, şifre güvenliği).
