# Proje Takip Sistemi v2.0.2

## Öne Çıkanlar

- Windows `.exe` içinde `Dokümanı Görüntüle` akışı daha güvenilir sistem açma yoluna taşındı.
- Preview/document-open akışı daha küçük servis ve helper katmanlarına ayrıldı.
- `services/report_service.py` içinde `pandas` lazy-load edildi.
- Boş `DejaVuSans.ttf` asset'i gerçek font dosyasıyla değiştirildi.

## Teknik Notlar

- Windows'ta belge açma için `os.startfile` önceliklendirildi.
- Preview payload, document lookup, preview state ve preview render hazırlığı ayrı katmanlara bölündü.
- PDF rapor font çözümleme artık gerçek `DejaVuSans.ttf` asset'i ile çalışıyor.
